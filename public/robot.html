ROBOTISATION EN OPEN SOURCE

Le projet en 3 étapes

Recherche du robot open source

Trouver un robot polyvalent aussi bien pour la préparation des commandes que pour l'adapter sur un ccentre d'usinage.

Développement de l'interface

Déveloper une interface afin de pouvoire être autonome sur la programmation du robot.

Mise en place dans le Stock

Utiliser le robot pour les préparations commandes clients et faire évoluer et optimiser le produit.
